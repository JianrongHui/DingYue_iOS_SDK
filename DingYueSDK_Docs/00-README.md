# DingYueSDK 私有化规格文档

本目录包含 DingYueSDK 私有化改造的可落地规范文档，面向后端、iOS、Web、数据与运营团队。

## 目录
- 01-Solution-Overview.md - 方案总览、范围与核心流程
- 02-Backend-APIs.md - 接口规范、鉴权、配置与事件接口
- 03-Database-Schema.md - D1 (SQLite) 数据表结构
- 04-Events-Dictionary.md - 事件字典、字段与错误码
- 05-Analytics-Mapping.md - GA4/Firebase 事件映射
- 06-Targeting-Experiments.md - 规则引擎与实验分流
- 07-Admin-Console.md - 管理后台页面与功能
- 08-H5-Package-Spec.md - H5 包结构、manifest 与桥接协议
- 09-Data-Warehouse.md - 数仓模型与 ETL
- 10-Analytics-SQL.md - 常用分析 SQL 模板
- 11-SDK-Event-Spec.md - iOS 侧事件字段规范
- 12-OpenAPI-Full.yaml - 完整 OpenAPI 草案
- 13-Operations-Security-QA.md - 运营、风控、监控与测试
- 14-Development-Execution-Plan.md - 开发执行方案（可交接）
- 15-Progress-Tracker.md - 开发进度追踪（已完成/待办）
